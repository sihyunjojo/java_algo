## 📝 File Naming Convention
문제 풀이 파일은 '플랫폼_레벨_문제_이름_메모.java' 형태의 네이밍 사용
```
ex)
Bj_S1_문제_이름_메모.java
Swea_d2_문제_이름_메모.java
```
